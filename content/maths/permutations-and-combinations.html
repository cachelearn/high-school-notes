Permutations & Combinations

1. Fundamental Principle of Counting

Multiplication Rule: If task A can be done in $m$ ways and task B in $n$ ways, together they can be done in $m \times n$ ways.

Addition Rule: If A or B (mutually exclusive), total $= m + n$ ways.

JEE Angle: Always identify whether tasks are done together (multiply) or alternatively (add). This underlies every P&C problem.
Example: How many 2-digit numbers can be formed using {1,2,3,4,5} without repetition?
Tens place: 5 choices  |  Units place: 4 choices  →  $5 \times 4 = \mathbf{20}$

2. Factorial

$$n! = n \times (n-1) \times (n-2) \times \cdots \times 1, \qquad 0! = 1$$

Key identity: $n! = n \times (n-1)!$ — used everywhere in P&C.

3. Permutation — $^nP_r$

Arrangement of $r$ objects from $n$ distinct objects where order matters.

$$^nP_r = \frac{n!}{(n-r)!}$$

JEE Angle: Words from letters, digits in numbers, rank of a word in dictionary.
Example: How many 3-letter words from the letters of "MASTER"?
$^6P_3 = 6 \times 5 \times 4 = \mathbf{120}$

4. Combination — $^nC_r$

Selection of $r$ objects from $n$ where order does NOT matter.

$$^nC_r = \frac{n!}{r!\,(n-r)!}$$

Key Properties

  • $^nC_r = {^n}C_{n-r}$
  • $^nC_r + {^n}C_{r-1} = {^{n+1}}C_r$   (Pascal's Identity)
  • $^nC_0 = {^n}C_n = 1$
  • $^nC_r = {^n}C_s \Rightarrow r = s$ or $r + s = n$
JEE Angle: Selection of teams/committees, handshakes, geometry (lines/triangles from points).
Example: Committee of 3 from 5 men and 4 women with at least 1 woman.
Total $-$ (no woman) $= {^9}C_3 - {^5}C_3 = 84 - 10 = \mathbf{74}$

5. Relation Between P and C

$$^nP_r = r! \times {^n}C_r$$

Permutation = Selection × Arrangement. Use this to switch between the two.

6. Box & Gap Method

  • Gap Method (no two together): Arrange the other objects first, then place restricted objects in the gaps created.
  • Tie Method (must be together): Treat the group as a single unit, arrange, then arrange within the group.
JEE Angle: "No two vowels together", "all consonants together" — very frequently tested.
Example: 4 boys and 3 girls in a row, no two girls adjacent.
Arrange 4 boys: $4! = 24$ → 5 gaps created → place 3 girls: $^5P_3 = 60$
Total $= 24 \times 60 = \mathbf{1440}$

7. Permutation of Alike Objects

When objects are not all distinct:

$$\text{Arrangements} = \frac{n!}{p!\, q!\, r!\, \cdots}$$

where $p, q, r, \ldots$ are frequencies of identical objects.

JEE Angle: Number of words from letters of a given word, arrangements with repeated elements.
Example: Arrangements of "MISSISSIPPI" (11 letters: M×1, I×4, S×4, P×2)
$$\frac{11!}{4!\,4!\,2!} = \mathbf{34650}$$

Example: Non-negative integer solutions to $x + y + z = 10$.

Example: 3-digit numbers from {1,2,3,4,5}, repetition allowed.

13. Number Formation & Divisibility

Key Rule: Always fix the constrained position first, then fill the rest.

When 0 is in the digit set — first digit cannot be 0, so treat it separately.

Example: 4-digit numbers from {0,1,2,3,4}, no repetition.
First digit: 4 choices (not 0) → remaining: $4 \times 3 \times 2 = 24$ → Total $= 4 \times 24 = \mathbf{96}$

Example: 4-digit numbers from {1,2,3,4,5,6} (no repetition), divisible by 2.
Last digit (even): 3 choices → remaining 3 positions: $5 \times 4 \times 3 = 60$ → Total $= 3 \times 60 = \mathbf{180}$

15. Grid Path Counting

From $(0,0)$ to $(m,n)$ moving only right or up — need exactly $m$ rights and $n$ ups:

$$\text{Total paths} = {^{m+n}}C_m = \frac{(m+n)!}{m!\, n!}$$

With obstacle at point $(a,b)$: subtract paths passing through it.

Example: Paths from $(0,0)$ to $(4,3)$ with obstacle at $(2,1)$.
Total $= {^7}C_4 = 35$
Through $(2,1)$: ${^3}C_2 \times {^4}C_2 = 3 \times 6 = 18$
Valid paths $= 35 - 18 = \mathbf{17}$

16. Miscellaneous

Lines & Triangles from Points

From $n$ points, no 3 collinear:   Lines $= {^n}C_2$,   Triangles $= {^n}C_3$

If $m$ points are collinear (among $n$):

$$\text{Lines} = {^n}C_2 - {^m}C_2 + 1 \qquad \text{Triangles} = {^n}C_3 - {^m}C_3$$

Among the $m$ collinear points, $^mC_2$ pairs all lie on the same single line — so instead of $^mC_2$ lines we count just 1 (hence $-{^m}C_2 + 1$). Collinear points form no triangle, so subtract $^mC_3$.
Example: 10 points, 4 collinear.
Lines $= {^{10}}C_2 - {^4}C_2 + 1 = 45 - 6 + 1 = \mathbf{40}$
Triangles $= {^{10}}C_3 - {^4}C_3 = 120 - 4 = \mathbf{116}$

Diagonals of a Polygon

In a polygon with $n$ sides, a diagonal connects two non-adjacent vertices.

$\text{Diagonals} = {^n}C_2 - n = \frac{n(n-1)}{2} - n = \frac{n(n-3)}{2}$

$^nC_2$ = all line segments between $n$ vertices. Subtract $n$ sides to get only diagonals.
Polygon$n$Diagonals
Quadrilateral42

Example: A polygon has 20 diagonals. Find the number of sides.
$\dfrac{n(n-3)}{2} = 20 \Rightarrow n^2 - 3n - 40 = 0 \Rightarrow (n-8)(n+5) = 0 \Rightarrow n = \mathbf{8}$

Rank of a Word in Dictionary

  1. Arrange all letters alphabetically.
  2. Count words starting with letters before the first letter of the given word.
  3. Repeat for each subsequent position.

Derangement

No object in its original position:

$$D_n = n!\sum_{k=0}^{n} \frac{(-1)^k}{k!}$$

Exponent of Prime $p$ in $n!$

$$E_p(n!) = \left\lfloor\frac{n}{p}\right\rfloor + \left\lfloor\frac{n}{p^2}\right\rfloor + \left\lfloor\frac{n}{p^3}\right\rfloor + \cdots$$

Example: Trailing zeros in $100!$ = $E_5(100!) = 20 + 4 = \mathbf{24}$
